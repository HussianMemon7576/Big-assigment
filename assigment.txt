Q9.
switch case statement kya hoti hai? break ka role explain karein.

Switch ka use jub karte hai jub aik hi value se hum khelna ho isme hum opreator ka use nhi karte bus variable banate hai and switch se compare karte hai 
Example
let color = "red";
switch(color){
    case "red":
        console.log("Stop");
        break;
    case "green":
        console.log("Go");
        break;
    case "yellow":
        console.log("Wait");
        break;
    default:
        console.log("Unknown color");
}

agar hum breake na lagaye to agle sare cases bhi chal jaenge sahi match hone ke bawajood is se js behaviour khrab hota hai
